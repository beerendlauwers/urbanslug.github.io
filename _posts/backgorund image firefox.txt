1. Get the stylish add on for firefox: https://addons.mozilla.org/en-US/firefox/addon/stylish/?src=search

2. In add ons page sidebar you will find user styles with the stylish logo. Click "Write new style."

This should be its content though (you can edit the css if you please):
``` css
 /* Set a wallpaper for the about:newtab and the about:home page*/
 @-moz-document url("about:newtab"), url("about:home") {
   /*for the new tab page*/
   #newtab-scrollbox {
     background: url("file:///home/urbanslug/pictures/wallpaper/marvel_comics_characters-wallpaper-1366x768.jpg") no-repeat !important ;
     background-size: 100% !important;
    }
   /*for the homepage*/
   body {
     background: url("file:///home/urbanslug/pictures/wallpaper/") no-repeat !important ;
     background-size: 100% !important;
   }
   #launcher {
     background-color: rgba(0,0,0,0.8) !important;
   }
 }
```
3. Save.

If you're wondering about the filename needing an extension, it doesn't.
